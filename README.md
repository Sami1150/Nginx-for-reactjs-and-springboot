# **The common commads:**
<br>
To goto directory of nginx:	cd C:\Others\nginx-1.24.0<br>

To run nginx: 			_./nginx.exe_<br>

To stop nginx: 			_nginx.exe -s stop_<br>

To see if it is running: 	_tasklist /fi "imagename eq nginx.exe"_<br>
